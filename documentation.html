Everything About Scrath

What is Scrath?

Scrath stands for Script-to-Math. You write logical math equations and Scrath makes it look like real math.

Note: you can find a table with all functions and macros at the end of the page, for quick reference

Simple Example

x/y

To use Scrath, go to the Scrath home page.

How it works

Scrath takes your input Scrath code and transforms it into LaTeX code, which is then interpreted by MathJax into nice math formulas.

Documentation and Tutorial

A lot of the time, you can just write math as you intuitively would, and it Scrath will work its magic.

Example

(a^b)/(lambda - 1)

Scrath separates your code words into 2 categories: functions and normal text.

How do '(' and ')' work?

When Scrath detects a function (such as 'pow', which is called like pow(x,2); it stands for 'x to the power of 2'), it replaces the whole 'pow(x,2)' structure with the actual value of the function, which is x^2, and it knows to transform it nicely into

Otherwise, it treats parenthesis as normal text.

Functions

A function (or a macro) is something that allows you to more easily write Scrath code.

Declaring / Creating a Function

function circleArea(r) pi r^2

This is how we write the area of a circle with radius 5:

circleArea(5)

In this example, 'r' is replaced with '5', wherever 'r' would be when the function is used

Hint: You can give multiple words as a single parameter when using a function! For example, sqrt(I have a line above my head)

If a function has no parameters, you can just call it without parenthesis (you can still use '()' if you want)

Parameters Example

function longSequence() 1+2+3+4+5+6+7+8+9

longSequence()

Is the same as

longSequence

You can call functions within functions or give them as parameters!

Note: To go to a new line in the output math, you must use 2 new lines in Scrath (press enter twice).

Functions Are Awesome

function sum(x,y) sqrt(x)+sqrt(y)

function fraction(x,y) x/y

sum(fraction(x,y), z)

Functions with Any Number of Parameters

You can create special functions that loop through the parameters and repeat the function for each parameter:

Normal Mode

function anyParams(...) [...^2]

anyParams(c,3,5,63)

We can also use the 'between' mode to put something between every repetition:

With 'between' Mode

function anyParams(between, ...) [...^2]

anyParams(+,c,3,5,63)
